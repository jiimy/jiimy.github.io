##### 다크 모드 및 테마 분류
```
@mixin dark-mode($type:dark-mode, $theme-type:basic){
  $color:'';
    // comment: 타입별 변경되는 것이 있을땐 여기에 변수 항상 지정해야 된다는 단점.
  @if ($type==dark-mode){
    $color: black;
  }
  @if ($type==light-mode){
    $color: white;
  }
}

.wrap {
  &.is-dark {
    @include dark-mode(dark-mode);
    // 타입과는 별도 구분인건 이곳에 작성
  }
  &.is-light {
    @include dark-mode(light-mode);
  }
}
```

아래의 map과 for을 이용해서 코드 리팩토링(약 25라인)
```
$class-img : (
        1: pale-blue,
        2: sky-blue,
        3: purple-blue,
        4: deeppurple-blue
      );
      $box-height : (
        1: 136px,
        2: 197px,
        3: 233px,
        4: 269px
      );
      @for $i from 1 through 4 {
        &.order-#{$i} {
          @include vw-convert(height, map-get($box-height, $i));
          @each $name, $color in $class-img {
            &.is-#{$color} {
                .choice-box {
                  background: url("components/imgs/box#{$i}_#{$color}.svg") no-repeat bottom;
                }
            }
          }
        }
      }
```
